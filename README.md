# FA BANKS PLC – Information Security Risk Assessment (GRC Project)

##  Overview
This project is a simulated Governance, Risk, and Compliance (GRC) assessment for a fictional bank (FA BANKS PLC). It demonstrates my ability to identify, assess, and document information security risks within a financial institution environment.



##  Project Context
FA BANKS PLC is a financial institution that depends on digital banking systems, customer data platforms, and third-party service providers. This assessment focuses on identifying risks that may affect the confidentiality, integrity, and availability of banking operations and information assets.



##  Project Contents
- Information Security Risk Assessment Report (PDF / Word)
- Risk Register (Excel)
- Supporting documentation (if applicable)



##  Scope of Assessment
This assessment covers:
- Core banking applications  
- Online and mobile banking platforms  
- Customer information systems  
- Internal employee systems  
- Network and infrastructure components  
- Third-party vendors supporting banking operations  


## Key Activities Performed
- Identified information security risks  
- Assessed risk likelihood and impact  
- Assigned risk ratings (Low, Medium, High, Critical)  
- Recommended security controls and mitigation strategies  
- Documented findings in formal GRC reporting format

- Risk identification and analysis  
- Risk assessment methodology
